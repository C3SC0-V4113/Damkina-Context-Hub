# Flujos

Esta carpeta documenta secuencias de usuario, negocio, sistema o proceso.

## Convención

- Archivo: `flujo-slug-descriptivo.md`
- Template: `99-plantillas/flujo.md`
- Tipo: `flujo`

## Qué Debe Incluir Un Flujo

- Objetivo.
- Actores.
- Precondiciones.
- Paso a paso principal.
- Variantes o errores.
- Resultado esperado.
- Contextos y decisiones relacionadas.

Si el flujo requiere una decisión arquitectónica nueva, crea o relaciona un ADR en `01-contextos/decisiones/`.
