# Contexto De Dominio

Esta carpeta describe conceptos, entidades, reglas y lenguaje del negocio.

## Convención

- Archivo: `YYYY-MM-DD-slug-descriptivo.md`
- Template: `99-plantillas/contexto.md`
- Tags habituales: `dominio`, `producto`, `arquitectura`

## Qué Documentar Aquí

- Entidades y conceptos de negocio.
- Reglas de dominio.
- Glosario.
- Casos especiales.
- Restricciones que afecten frontend, backend o datos.
