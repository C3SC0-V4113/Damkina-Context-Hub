# Contexto De Producto

Esta carpeta guarda objetivos, alcance, usuarios, necesidades, restricciones y decisiones funcionales del producto.

## Convención

- Archivo: `YYYY-MM-DD-slug-descriptivo.md`
- Template: `99-plantillas/contexto.md`
- Tags habituales: `producto`, `ux`, `dominio`

## Qué Documentar Aquí

- Objetivos de producto.
- Perfiles de usuario.
- Alcance funcional.
- Reglas de priorización.
- Restricciones de negocio.
- Criterios de éxito.
