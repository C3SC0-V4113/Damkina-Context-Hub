# Contexto De Operaciones

Esta carpeta guarda información sobre ambientes, despliegue, observabilidad, soporte y mantenimiento.

## Convención

- Archivo: `YYYY-MM-DD-slug-descriptivo.md`
- Template: `99-plantillas/contexto.md`
- Tags habituales: `operaciones`, `seguridad`, `performance`, `arquitectura`

## Qué Documentar Aquí

- Ambientes.
- Estrategia de despliegue.
- Monitoreo y logging.
- Backups y recuperación.
- Procedimientos de soporte.
- Restricciones operativas.
