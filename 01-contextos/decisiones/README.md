# Decisiones ADR

Esta carpeta guarda decisiones significativas del proyecto. Cada decisión debe explicar el contexto, las alternativas evaluadas, la decisión tomada y sus consecuencias.

## Convención

- Archivo: `NNNN-slug-decision.md`
- Template: `99-plantillas/decision-adr.md`
- Tipo: `decision`
- Estados habituales: `propuesto`, `aceptado`, `deprecated`, `superseded`

## Cuándo Crear Un ADR

Crea un ADR cuando se defina o cambie una decisión sobre arquitectura, frontend Flutter, backend, persistencia, APIs, seguridad, integraciones, despliegue u operación.

## Mantenimiento

Actualiza `00-indices/decisiones.md` y `00-indices/contextos.md` cada vez que se agregue o cambie un ADR.
