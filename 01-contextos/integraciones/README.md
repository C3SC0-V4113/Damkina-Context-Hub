# Contexto De Integraciones

Esta carpeta documenta servicios externos, APIs de terceros, webhooks, sincronización y contratos entre sistemas.

## Convención

- Archivo: `YYYY-MM-DD-slug-descriptivo.md`
- Template: `99-plantillas/contexto.md`
- Tags habituales: `integracion`, `api`, `backend`, `seguridad`

## Qué Documentar Aquí

- Sistemas externos.
- Credenciales o modelos de autenticación, sin secretos reales.
- Contratos y payloads relevantes.
- Frecuencia de sincronización.
- Fallos esperados y manejo de errores.
- Dependencias con frontend, backend o dominio.
