# Referencia: Diseño Figma Mobile

## Fuente

- URL: https://www.figma.com/design/GkdRWwyzqC2B86yO3jn335/Proyecto-Damkina?node-id=0-1&p=f&t=i1nfAJkcuLLg0zXd-0
- Archivo observado: Proyecto Damkina
- Nodo observado: `0:1`
- Fecha de consulta: 2026-04-21

## Resumen

Diseño mobile del MVP de Damkina. Sirve como fuente visual principal para composición, jerarquía, pantallas y estilo del frontend Flutter.

## Pantallas Observadas

- Login simple.
- Onboarding de nombre.
- Onboarding de ubicación con mapa.
- Agregar ubicación.
- Editar ubicación.
- Recomendaciones o home de cultivos.
- Detalle de cultivo.
- Mis ubicaciones.
- Perfil.
- Navegación inferior móvil.

## Ideas Relevantes Para Damkina

- El diseño está enfocado en mobile.
- La UI usa una estética agrícola/natural con tarjetas, mapas, imágenes de cultivos y navegación inferior.
- Las pantallas de mapa usan pin central, controles flotantes y tarjeta inferior para nombrar ubicación.
- El detalle de cultivo prioriza imagen, explicación contextual, factores de crecimiento y recomendaciones.
- La navegación inferior sugiere secciones principales: cultivos, ubicaciones y perfil.

## Limitaciones

- No hay diseño específico documentado para tablets o dispositivos intermedios.
- En la inspección actual no estuvieron disponibles variables de diseño desde Figma.
- Code Connect no estuvo disponible por limitaciones de plan/permisos.
- Los tokens de Flutter deben derivarse manualmente del diseño hasta que exista una fuente formal de variables.
