# Referencias

Esta carpeta guarda fuentes externas, artículos, documentación, notas de investigación y enlaces relevantes.

## Convención

- Archivo: `referencia-slug-descriptivo.md`
- Template: `99-plantillas/referencia.md`
- Tipo: `referencia`

## Qué Documentar Aquí

- Artículos.
- Documentación oficial.
- Repositorios externos.
- Investigaciones técnicas.
- Comparativas de tecnologías.

Una referencia no es una decisión. Si una referencia lleva a elegir un camino, registra la decisión en `01-contextos/decisiones/`.
