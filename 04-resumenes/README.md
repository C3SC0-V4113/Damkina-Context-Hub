# Resúmenes

Esta carpeta guarda síntesis de contexto para onboarding, revisión rápida o estado actual de un área.

## Convención

- Archivo: `resumen-slug-descriptivo.md`
- Template: `99-plantillas/resumen.md`
- Tipo: `resumen`

## Qué Debe Incluir Un Resumen

- Alcance de la síntesis.
- Contexto vigente.
- Decisiones importantes.
- Riesgos o vacíos.
- Links a documentos fuente.

Los resúmenes no reemplazan a los ADRs ni a los contextos fuente.
