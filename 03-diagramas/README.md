# Diagramas

Esta carpeta guarda diagramas en Markdown, Mermaid o referencias a archivos externos.

## Convención

- Archivo: `diagrama-slug-descriptivo.md`
- Template: `99-plantillas/diagrama.md`
- Tipo: `diagrama`

## Formatos Recomendados

- Mermaid para diagramas editables en Markdown.
- Links a imágenes o herramientas externas cuando el diagrama no pueda mantenerse en texto.
- Descripción textual para que agentes puedan interpretar el diagrama aunque no rendericen imágenes.

Todo diagrama debe indicar qué contexto o decisión representa.
